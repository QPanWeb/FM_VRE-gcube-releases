Usage
------------------------------------------------------------


<HOST>/images/report 		: shows current status report

<HOST>/images/report/xml	: shows current status summary in xml 

<HOST>/images/help			: shows this page
		
<HOST>/images/pics?species=<SCIENTIFIC_NAME>	: retrieves the picture of specified species

<HOST>/images/maps/<ALGORITHM>?species=<SCIENTIFIC_NAME>&product=<PRODUCT_TYPE>	: retrieves the <PRODUCT_TYPE> map of the specified species distribution according to specified algorithm. 

<HOST>/images/maps/<HSPEC_ID>?species=<SCIENTIFIC_NAME>&product=<PRODUCT_TYPE>	: retrieves the <PRODUCT_TYPE> map of the specified species distribution generated using the specified HSPEC table.


Where:

<HSPEC_ID> the id of a HSPEC table 

<PRODUCT_TYPE> must be one of the following : 

		IMAGE
		GIS
		
if argument product is not specified, the default value IMAGE is used. 


<ALGORITHM> must be one of the following :

	native
	native/2050
	suitable
	suitable/2050
	
	
	
<SCIENTIFIC_NAME> case insensitive species' scientific name without spaces eg:



Sphyraenops Bairdianus -> SphyraenopsBairdianus (equivalent to sphyraenopsbairdianus)
