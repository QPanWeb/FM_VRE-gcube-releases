Service Profile - AnySoftware Data

Software Data

Please provide the required software information:

  • Name: A name associated to the software to be registered. The provided value will be used for both the generated service and package.
  • Description: A short description of the software. The provided value will be used for both the generated service and package.
  • Version: The software version. The provided version number will be assigned to the newly generated package. A version number of '1.0.0' will be assigned to the service.
* Mandatory field.
