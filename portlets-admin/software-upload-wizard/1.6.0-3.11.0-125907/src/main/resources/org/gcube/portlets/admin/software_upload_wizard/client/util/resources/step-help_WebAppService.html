Service Profile - Service Data

Third Party selection

If the software is from third party please select this checkbox.
Third party software registration is not allowed if you are working on the 'gcube' scope infrastructure.

Service Data

In this section the user enters data related to Service section of the Service Profile. All fields are mandatory if not otherwise declared.

  • Service name: Name assigned to the Service in the Service Profile. Only alphanumeric chars and '-' symbol are allowed.
  • Service description (optional): Description assigned to the Service in the Service Profile.
  • Service version: Version assigned to the Service in the Service Profile.
  • Service class: Class assigned to the Service in the Service Profile. The class may be assigned by the system for certain software types. Only alphabetical chars and '-' symbol are allowed.
* Mandatory field.
