Software Registration Data

Please provide the Maven coordinates of the artifact you want to register on the gCube infrastructure:

  • Artifact Id: the artifact id associated to the artifact to be registered.
  • Group Id: the artifact group id.
  • Artifact Version: the artifact version.
  • Snapshot: tick the checkbox only if the artifact to be registered is hosted on a maven snapshot repository.
  • Jar archive filename: If the artifact is hosted on a snapshot repository the filename of the jar primary artifact must be provided.

Only jar packaged maven artifacts can be registered using this wizard.

* Mandatory field.
