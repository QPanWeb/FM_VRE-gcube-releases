Package Maven Artifact Coordinates

Artifact Coordinates

Please specify in this section the coordinates related to the main artifact for the package. All fields are mandatory.

  • Group Id: the Group Id related to the Maven artifact. Only alphanumerical chars and '-','_','.' symbols are allowed.
  • Artifact Id: the Artifact Id related to the Maven artifact. Only alphanumerical chars and '-' symbol are allowed.
  • Artifact Id: the Group Id related to the Maven artifact. A valid Maven version is required (e.g.: 1.0.0 or 1.2.3-SNAPSHOT).
* Mandatory field.
