Service Profile Data

Software Data

Please provide the required information regarding the software package to be registered:

  • Name: A name associated to the software to be registered. The provided value will be used for both the generated service and package.
  • Description: A short description of the software. The provided value will be used for both the generated service and package.
  • Version: The software version. The provided version number will be assigned to the newly generated package. A version number of '1.0.0' will be assigned to the service.
  • Service Class: The service class that will be assigned to the generated service in the service profile.

Target Service

Please provide required information regarding the plugin targeted service and package:

  • Service Class: The service class associated to the target service.
  • Service Name: The name associated to the target service.
  • Service Version: The version associated to the target service.
  • Package Name: The package name associated to the target service package.
  • Package Version: The package version associated to the target service package. Can include "-SNAPSHOT" suffix.
* Mandatory field.
