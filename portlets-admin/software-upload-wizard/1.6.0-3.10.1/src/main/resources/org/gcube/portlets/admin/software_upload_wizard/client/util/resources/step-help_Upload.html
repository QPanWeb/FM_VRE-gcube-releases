Upload package files

File selection

Please select the file type from the available file types combobox and select the file on your local system by pressing the "Browse..." button.
After pressing the "Upload" button the selected file will be uploaded and checked against file type specific rules. Any problem regarding the uploaded file will be notified.
In order to proceed to the next step the user must provide at least one file for each mandatory file type.

Uploaded Files Table

The table shows uploaded files which have passed file type validation process.
Is it possible to remove entries by selecting multiple rows on the table and clicking on the remove button.

Allowed file types

Availability of a file type depends on the service package, please check the list of allowed file types and file type rules before uploading any file.
